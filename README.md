# Formulario de Registro Inteligente
Formulario de registro e inicio de sesión con validación de contraseñas, almacenamiento en localStorage y diseño responsivo.

Mi primer formulario con login y registro hecho con HTML, CSS y JavaScript puro.
Lo empecé en clase y fui agregándole cosas por mi cuenta.

## ¿Qué hace?

- Registro de usuarios con validación de contraseña
- Inicio de sesión con autenticación
- Guarda los usuarios en localStorage (sin base de datos)
- Medidor de seguridad de contraseña
- Mostrar y ocultar contraseña
- Diseño responsivo con animación de panel deslizante

## Tecnologías

- HTML5
- CSS3
- JavaScript
